# Source Attribution

This document tracks the origin and licensing of third-party code, designs, or assets integrated into Verbum Minecraft, particularly those adapted from open-source projects or inspired by existing works. This ensures compliance with licenses and provides a transparent record for clean-room development.

---

## Instructions for Contributors (especially AI Agents):

*   **Before Importing:** Always check if the functionality or design can be implemented from scratch (`clean-room`) based on high-level specifications.
*   **If Importing/Adapting:**
    1.  Place original third-party source code in the `/third_party/` directory, maintaining its original directory structure where feasible.
    2.  Include a copy of the associated license in the `/LICENSES/` directory.
    3.  Create an entry in this `SOURCE_ATTRIBUTION.md` document.
    4.  If a design is heavily inspired by an external project but reimplemented, document the inspiration here, citing the original source.
*   **Update Regularly:** This document must be updated any time external code, assets, or significantly inspired designs are incorporated or modified.
