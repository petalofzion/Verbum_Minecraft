# Source Attribution

This document tracks the origin and licensing of third-party code, designs, or algorithms integrated into Verbum Minecraft.

## Policy

**Stricter Attribution:**
- **Code:** Any code copied or adapted from external projects must be attributed.
- **Design/Algorithm:** If an implementation closely follows a specific design or algorithm from an external project (even if rewritten from scratch), it must be attributed.
